双指针覆盖算法可视化

函数 list_delete_elem(List L, ElemType e) 会在一次扫描中删除所有值等于 e 的元素。核心思想很直接：i 负责读取原数组， k 负责把需要保留的元素写回前部。

算法

void list_delete_elem(List L, ElemType e) {
  if (L == NULL) return;
  int k = 0;
  for (size_t i = 0; i < L->length; ++i) {
    if (L->data[i] != e) {
      L->data[k++] = L->data[i];
    }
  }
  L->length = k;
}
  • 原理：每个需要保留的元素都会被复制到当前最靠前的可用位置，因此有效元素之间不会出现空洞。
  • 时间复杂度：O(n)。整个数组只扫描一次。
  • 空间复杂度：O(1)。算法原地修改数组，只使用两个下标。
  • 设计要点：这是一个稳定过滤（stable filter）过程，保留元素的相对顺序不会改变。
